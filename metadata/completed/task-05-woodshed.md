# Task 5: The Woodpile & Woodshed

## User Story
As a player, I want to build a Woodpile or Woodshed so that I can season my firewood significantly faster than leaving it in a stockpile.

## Acceptance Criteria
- [ ] `Woodpile` building defined and constructible from Green Firewood.
- [ ] `Woodshed` storage building defined (requires Complex Furniture).
- [ ] Seasoning speed multiplier (1.5x) applied when wood is in a Woodpile.
- [ ] Seasoning speed multiplier (2.5x) applied when wood is in a Woodshed.
