# Task 2: Basic Fuel Materials (XML)

## User Story
As a player, I want to see distinct item entries for Green Firewood, Seasoned Firewood, and Charcoal in my storage filters so I can manage my fuel reserves.

## Acceptance Criteria
- [ ] `ThingDef` for Green Firewood exists.
- [ ] `ThingDef` for Seasoned Firewood exists.
- [ ] `ThingDef` for Charcoal exists.
- [ ] All fuels are grouped under a new `ThingCategoryDef` (RimWood_Fuel).
- [ ] Materials have appropriate weights and stack sizes per the design doc.
- [ ] Market values reflect labor/time (Seasoned > Green > Raw).
- [ ] Flammability is correctly set to match vanilla wood (1.0).
